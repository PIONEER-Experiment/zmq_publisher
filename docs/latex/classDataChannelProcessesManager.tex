\doxysubsection{Detailed Description}
Manages data channel processors and their execution. 

The {\ttfamily \mbox{\hyperlink{classDataChannelProcessesManager}{Data\+Channel\+Processes\+Manager}}} class is responsible for managing a collection of data channel processors and coordinating their execution. It also maintains a data buffer to store the output generated by the processors.
