\doxysubsection{Detailed Description}
A utility class for executing commands and managing execution parameters. 

The {\ttfamily \mbox{\hyperlink{classCommandRunner}{Command\+Runner}}} class provides functionality to execute commands with optional arguments and manage execution parameters such as wait time.
