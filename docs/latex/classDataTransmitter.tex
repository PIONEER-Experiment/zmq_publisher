\doxysubsection{Detailed Description}
Transmits data over a Zero\+MQ (zmq) publisher socket. 

The {\ttfamily \mbox{\hyperlink{classDataTransmitter}{Data\+Transmitter}}} class provides functionality for binding to a zmq publisher socket and publishing data to a specific zmq-\/address.
